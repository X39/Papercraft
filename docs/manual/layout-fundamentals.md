## What Is This?

Layout fundamentals explain how space is measured and used while the template is rendered.
This includes available space, margins, padding, border thickness, alignment, length units, colors and text styling.

## When Should I Use This?

Use this chapter when content is too large, too small, misaligned, too close to other content,
or visually unclear.
It should help you choose between margin, padding and border settings before changing many controls.

## How Do I Start?

Start with one visible layout change at a time.
For example, use a `border` around one piece of content before applying the same idea to a larger section.
